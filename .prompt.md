Modules:

/ (Home Page)
    A central landing page that directs users to the appropriate portal.
    - Features a card for the "Admin Portal" which links to /admin/login.
    - Features a card for the "Student Portal" which links to /exam/login.

/admin/login
    Login page for administrators. After successful login, redirects to the admin dashboard.

/admin/ (protected)
    The main admin interface with a side navigation menu.
 
/admin/dashboard
    Displays key statistics about the platform.

/admin/skills
    Allows admins to create and manage skills. For each skill, admins can add, view, and manage questions. The question creation form dynamically adapts to the question type (multiple choice, free text, or code), including a detailed test case editor for coding questions.

/admin/students
    Allows admins to view the list of all students and upload new students via a CSV file.

/admin/assessments
    Admins can create new assessments, defining their name, duration, and the skills/number of questions involved. Existing assessments are also listed.

/admin/assignments
    Admins can create assignments based on existing assessments and assign them to students, with options to filter students by year of passing or academic stream. Details of assigned students and their unique access codes can be viewed.

/exam/login
    Login form for students to begin an assessment.
    - Requires selecting the assignment, entering an email, and a unique access code.

/exam/assignment/:assignmentStudentId (protected)
    The exam-taking interface for students.
    - Displays one question at a time with navigation controls (Previous/Next).
    - Shows a countdown timer, and auto-submits on timeout.
    - Answers are saved immediately.
    - A "Finalize" button allows the student to submit the exam manually.
